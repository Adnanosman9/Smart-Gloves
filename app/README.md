# SIGNLINK Mobile App

A React Native app that connects to both SIGNLINK smart gloves via Bluetooth, runs an on-device ML model to recognize BdSL gestures, and speaks the translation aloud.

---

## Features

- Connects to both left and right gloves simultaneously over BLE
- Live flex sensor visualizer for all 10 fingers
- On-device Random Forest gesture classifier (no internet needed)
- Text-to-speech output for every recognized gesture
- Sentence builder — strings multiple gestures into a full sentence
- Auto-speak toggle
- Connection status with signal strength (RSSI)

## Setup

### 1. Install dependencies

```bash
cd app
npm install
```

### 2. Android permissions

The app requires the following permissions (already configured in the manifest):
- `BLUETOOTH_SCAN`
- `BLUETOOTH_CONNECT`
- `ACCESS_FINE_LOCATION`

### 3. Add the ML model

The file `src/ml/modelWeights.js` must be generated by training a Random Forest on your gesture dataset.

Once you have collected data using the `data_collect` firmware:

```bash
cd ml
python train.py          # trains the model
python export_model.py   # exports modelWeights.js
```

Copy the output `modelWeights.js` into `src/ml/`.

### 4. Run the app

```bash
npx react-native run-android
# or
npx react-native run-ios
```

## How It Works

1. App scans for BLE devices named `SIGNLINK-L` and `SIGNLINK-R`
2. Connects to both and subscribes to their BLE notification characteristic
3. Each glove sends a JSON packet at 20Hz:
   ```json
   {"h":"L","f":[t,i,m,r,p],"a":[ax,ay,az],"g":[gx,gy,gz],"t":millis}
   ```
4. App combines both gloves into a 16-value feature vector and feeds it into the ML model
5. The model returns a gesture label and confidence score
6. If confidence > 72%, the gesture is displayed and spoken aloud

## ML Model

The gesture classifier is a **Random Forest** model exported from Python (`scikit-learn`) into a JavaScript object. It runs entirely on-device with no internet connection required.

**Input:** 16 floats — `[L_flex×5, L_accel×3, R_flex×5, R_accel×3]`

**Output:** gesture label + confidence score

The model uses a **sliding window** of 6 consecutive frames and a **confidence threshold of 72%** to avoid false positives.

---

## ⚠ Important Note

The app will not translate gestures until `src/ml/modelWeights.js` exists. Without it the app will crash on import. Generate this file by training the model on your dataset first.
